## Docker and Hydra

```sh
# 1 .docker build
docker build \
  -f dockerfiles/train.dockerfile \
  -t vdm-pokemon-train .

## 2. Docker Run
docker run --rm \
  -e WANDB_API_KEY=$WANDB_API_KEY \
  vdm-pokemon-train

## 3. Hydra Train
#### #Change batch size and learning rate
sh python -m vdm_pokemon.HydraTrain batch_size=128 learning_rate=1e-3 wandb.run_name=unet_bs128_lr1e-3
